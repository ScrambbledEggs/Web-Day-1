what is the internet?

The internet is a vast network of interconnected computers that communiate using standardized protocols. It allows users to access and share information globally.

HTML

what is HTML?

The first webpage was written in HTML (hyper text markup language). It is the standard mark up language used to create web pages. It structures content on the web, allowing browsers to display images, text, and other elements

How to use HTML?

HTML uses tags to define elements. Tags are enclosed in angle brackets. Elements can include headings, paragraphs, links, images, and lists.

HTML documents are structured with a basic layout, including a HEAD section for metadata and a BODY section for content
